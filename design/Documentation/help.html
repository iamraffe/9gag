1. HTML Structure

This theme was built with the help of the Bootstrap 3 framework.

Open the Aruna folder and your will find all HTML, CSS, JS and PHP files that you'll need to customize and add your content.



2. CSS Files and Structure

This template uses 8 CSS files, which can be found in the /css/ folder. For modifying the theme appearance, please modify just the style.css and add all of your custom code there. I have included both the less versions and the minified versions of the css files used.


If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited. If you're using firefox I'd suggest using Firebug, it's really awesome. Get firebug

3. Javascript

This template uses 13 different javascript files. Most of them are custom libraries needed within the theme, if you'd like to customize the theme or see how those libraries are used, edit only the js/main.js file, you shouldn't edit any of the other files.

4. Fonts

This theme uses the following fonts: Nunito and Dosis

5. File upload system

I have enabled basic front-end functionality for the front-end submission that includes a file upload system as well for images. It does NOT contain server-side functionality and you can't create a dynamic website as this is a HTML template, however, server-side functionality can be added easily I think, this jQuery plugin was used https://github.com/blueimp/jQuery-File-Upload, you can check their page for examples and demo.
